# Atmospheric CO2: Time Series Analysis & Forecasting

### [View webpage here](https://leonardo-p-taylor.github.io/CO2_Time_Series_Analysis/)

## Project Overview
This project focuses on the historical atmospheric carbon dioxide concentrations recorded at the Mauna Loa Observatory in Hawaii from 1959 to 1997 and forecasts. This was for a Time Series module at QMUL, making use of time series and Meta's Prophet to analyse and forecast the data.

## Files here

* 'README.md': Provides context to project and a link to the webpage.
* `index.html`: The live webpage hosted using Github Pages.
* `CO2_Analysis_Report.Rmd`: The R Markdown file containing all code and commentary.
* `CO2_Analysis.Rproj`: The R Project file.
